Control de flujo y manejo de errores

    • if...else: Estructura que permite tomar decisiones basadas en una condición.
    • switch: Estructura que evalúa una expresión y ejecuta diferentes bloques de código según su valor.
    • try/catch/throw: Mecanismo de manejo de errores. El bloque try ejecuta un código, el bloque catch captura y maneja los errores, y throw se utiliza para lanzar errores explícitamente.
    • Objetos Error: Representan los errores en JavaScript y proporcionan información sobre el error ocurrido.

Bucles e iteración

  • for: Ejecuta un bloque de código un número específico de veces.
  • while: Ejecuta un bloque de código mientras se cumple una condición.
  • do...while: Ejecuta un bloque de código al menos una vez y luego verifica una condición.

Expresiones y operadores

  • Asignación y Comparación: Permite asignar valores y compararlos para realizar evaluaciones.
  • Operadores aritméticos: Realizan operaciones matemáticas como suma, resta, multiplicación, etc.
  • Operadores lógicos: Permiten realizar operaciones lógicas como AND, OR, NOT, etc.

Números y fechas

  • Objeto Number: Proporciona funcionalidades relacionadas con números.
  • Objeto Math: Proporciona funcionalidades matemáticas y operaciones avanzadas.
  • Objeto Date: Permite trabajar con fechas y horas.

Colecciones indexadas

  • Arreglos: Permiten almacenar y acceder a varios valores en una sola variable.
  • Arreglos tipados: Especifican el tipo de datos de los elementos en el arreglo.

DOM (Document Object Model)

  • Encontrar elementos HTML por ID: document.getElementById()
  • Encontrar elementos HTML por nombre de etiqueta: document.getElementsByTagName()
  • Encontrar elementos HTML por nombre de clase: document.getElementsByClassName()
  • Encontrar elementos HTML mediante selectores CSS: document.querySelector() y document.querySelectorAll()
  • Encontrar elementos HTML por colecciones de objetos HTML: document.getElementsByTagName()
